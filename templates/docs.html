📘 Documentation Kirnos

📌 À quoi ça sert ?

Kirnos est une solution avancée de protection pour les lives TikTok. Il détecte automatiquement les comportements toxiques, les insultes et les raids. L’objectif est simple : sécuriser ton live et améliorer ton expérience.

⚙️ Comment ça marche ?

1. Tu connectes ton compte TikTok 2. Kirnos analyse les messages en temps réel 3. Les messages toxiques sont automatiquement bloqués 4. Tu gardes un contrôle total via le dashboard

💎 Premium

Le mode Premium débloque des fonctionnalités avancées : - Protection IA renforcée - Anti-raid intelligent - Dashboard complet avec statistiques - Support prioritaire

🚀 Pourquoi utiliser Kirnos ?

✔ Gain de temps ✔ Protection automatique ✔ Meilleure expérience pour tes viewers ✔ Outil professionnel pour streamer sérieusement
